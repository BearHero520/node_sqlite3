# node_sqlite3
一个简单的nodejs 服务 使用了express uuid sqlite3 做到 读写数据库 上传文件
默认访问端口 http://localhost:3000/



## 安装

>node 版本>=18.x

### 创建项目

通过 `npm init` 命令我们可以创建上述的任意模版

```bash
$ npm install #  也可以  npm i  
$ npm run start
$ open http://localhost:3000 # 访问应用
$ npm run pkg # 资源构建
```
